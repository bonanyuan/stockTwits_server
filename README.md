# StockTwits Server

## Dependency

Install dependency to your computer ,
```
    cd stockTwits_server
    virtualenv venv
    source venv/bin/source
    pip install -r requirements.txt
```
## Run the server

Run in terminal
```
    python test.py
    python server.py
```
and open http://127.0.0.1:5000/ in the browser.
