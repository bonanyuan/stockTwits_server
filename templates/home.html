Usage:

GET /sectors?sector=Technology

Filter S&P 100 symbols based on sectors. By default is set to give back all the S&P 100 stocks.
For the full list of S&P 100 symbols and their sectors, read the table S&P 100 Symbols

GET /search?symbol=AAPL

Search with symbol name and return the general information.

GET /twits?symbol=AAPL

For the selected symbol, return the newest 30 twits from StockTwits API

GET /sparkline?symbol=AAPL

For the selected symbol, return the recent 30 records of price.

GET /nlp?symbol=AAPL

For the selected symbol, return the 25 most common words.
